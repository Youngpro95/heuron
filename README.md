## 프로젝트 구조

```
- src/components: 공통 컴포넌트
- src/hooks: 커스텀 훅
- src/pages: 페이지
```

## 주요 기능

1. 과제 1

   - 이미지 확대/축소 (마우스 드래그)
   - 이미지 회전 (우클릭 드래그)
   - 그레이스케일 모드
   - 초기화 기능

2. 과제 2

   - 플레이어 수 설정
   - 카드 수 설정
   - 게임 진행 및 결과 확인

3. 과제 3
   - 모음/자음 분리 기능

## 기술 스택

- React
- TypeScript
- React Router
- Tailwind CSS

## 실행 방법

1. 프로젝트 클론
2. git clone https://github.com/Youngpro95/heuron.git
3. cd heuron-frontend
4. yarn install
5. yarn start
